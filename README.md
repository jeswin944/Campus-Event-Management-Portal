# Campus-Event-Management-Portal
A Flask-based Campus Event Management Portal that streamlines event registration, QR-code attendance tracking, certificate generation, notifications, on-duty management, academic timetable integration, and AI-powered event recommendations for students, faculty, and administrators.

Campus Event Management Portal is a comprehensive web-based platform developed using Flask and MySQL to simplify event organization, attendance management, certificate issuance, academic coordination, and student engagement within educational institutions.

The system provides dedicated dashboards for Administrators, Faculty Coordinators, and Students while integrating QR-based attendance, automated notifications, on-duty request management, timetable scheduling, exam scheduling, and AI-powered event recommendations.
